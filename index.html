Research Project

Optimizing the Early Detection of Dementia by Tracking the Progression of Parkinsons Disease Using Deep Learning and Computer Vision

  • The project involves selecting cohorts of Parkinsons patients with and without dementia, along with healthy controls, using PPMI clinical data and extracting relevant labels. A Python environment is set up and the MRI and clinical data are organized in a structured directory. An image-viewing pipeline is developed to load and visualize structural MRI scans. Two models are built in a cascading network, one to extract features from longitudinal MRI data to model PD progression, and another to predict dementia using progression scores and clinical features. The models are evaluated using metrics like accuracy, F1-score, and ROC-AUC, with Grad-CAM and SHAP employed for interpretability.
